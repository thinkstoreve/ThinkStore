ThinkStore Enterprise V9.5 PWA Descargable

Incluye:
- Página /install y /app para instalar Enterprise como aplicación.
- Botón flotante "Instalar ThinkStore Enterprise".
- Manifest mejorado con iconos, shortcuts y launch handler.
- Service Worker V9.5 con cache offline y actualización automática.
- Compatibilidad iPhone/iPad/Android.
- Pantalla offline.

No modifica:
- Resend
- Correos automáticos
- Recuperación de contraseña
- Pedidos
- Comprobantes
- Notas de entrega
- Panel actual de administración ThinkStore

Uso:
1. Subir el ZIP a GitHub como las versiones anteriores.
2. Esperar deploy de Netlify.
3. Abrir https://enterprise.thinkstore.com.ve/app o https://enterprise.thinkstore.com.ve/install.
4. En iPhone/iPad: Safari > Compartir > Añadir a pantalla de inicio.
5. En Android: Chrome > Instalar aplicación.
