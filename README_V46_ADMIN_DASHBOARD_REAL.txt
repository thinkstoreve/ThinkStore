ThinkStore V46 - Dashboard Profesional conectado a Supabase

Incluye:
- Admin lee clientes reales desde la tabla clientes de Supabase.
- Admin lee pedidos reales desde pedidos + pedido_items.
- Dashboard con KPIs reales: clientes, pedidos, preórdenes y ventas USD.
- Buscador de clientes y pedidos.
- Botones copiar correo/teléfono.
- Cambio de estatus intenta actualizar Supabase.
- Botón de actualización desde el dashboard.

Tablas usadas:
- clientes
- pedidos
- pedido_items

Nota:
Si Supabase no responde, el panel usa respaldo local para no romper la tienda.
