ThinkStore V7 - Panel administrativo y clientes

Incluye:
- Registro de clientes.
- Panel privado demo: botón Admin, clave [CODIGO_ADMIN].
- Control de clientes registrados.
- Control de pedidos y preórdenes.
- Cambio de estatus del pedido.
- Nota de entrega automática.
- Consulta de estatus por código de orden.
- Botón para enviar estatus por correo usando mailto.
- Masificación por correo usando BCC en la app de correo.
- Exportación CSV de clientes y pedidos.

Importante:
Esta versión guarda información en localStorage del navegador para pruebas en Netlify.
Para producción real se debe conectar Supabase para base de datos y un servicio de correo como Resend, SendGrid o Zoho/SMTP para envíos automáticos reales.
