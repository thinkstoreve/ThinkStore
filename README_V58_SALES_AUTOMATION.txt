ThinkStore V58 Sales Automation

Funciones agregadas/preparadas:

1. WhatsApp inteligente
- Mensajes automáticos por producto.
- Base para guardar leads y clientes interesados.
- Mensaje sugerido con producto, color y capacidad.

2. Cotizador PDF automático
- Código de cotización.
- Vigencia de oferta.
- Logo ThinkStore.
- Base para enviar cotización por correo o WhatsApp.

3. Mapa visual de preórdenes
- Pedido recibido.
- Comprado.
- En tránsito.
- Aduana.
- Venezuela.
- Disponible.
- Entregado.

4. Dashboard ejecutivo
- Ventas del día.
- Ventas del mes.
- Clientes nuevos.
- Preórdenes activas.
- Productos más vendidos.

5. Programa VIP ThinkStore
- Niveles Silver, Gold y Platinum.
- Base para beneficios, puntos extra y preventas.

6. Galería de videos
- Base visual para reviews, unboxing y videos de producto.

Nuevas funciones Netlify:
- quote-pdf.js
- whatsapp-leads.js
- executive-dashboard.js
- vip-program.js
- preorder-map.js

Nota:
Esta versión prepara la arquitectura para conectar estos módulos a Supabase, Resend y WhatsApp Business sin romper el flujo actual de pedidos.
