ThinkStore Service Center - Responsive + Bitácora

Cambios aplicados:
- Se mantiene la interfaz actual y el fondo actual.
- Se mejora la adaptación para PC, tablet y móvil.
- Se agrega módulo 'Bitácora' al menú.
- Se agrega botón 'Bitácora' dentro de cada orden.
- La bitácora permite registrar:
  diagnóstico, pruebas, presupuesto, repuesto, seguimiento, entrega y nota interna.
- Cada entrada actualiza el estado de la orden si el código coincide.
- Por ahora la bitácora guarda localmente en localStorage, igual que las órdenes actuales.
- La autenticación Supabase V2 se mantiene intacta.

Archivos modificados:
- index.html
- app.js
- styles.css

Subir estos archivos a GitHub y hacer deploy en Netlify.
