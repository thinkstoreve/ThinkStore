ThinkStore V10 - Panel Administrador Completo Demo

Clave admin: [CODIGO_ADMIN]

Incluye:
- Dashboard con métricas principales.
- Gestión de productos.
- Inventario por SKU, modelo, color y capacidad.
- Control de pedidos y estatus.
- Pagos con referencia y aprobación.
- Clientes registrados.
- Preórdenes.
- Envíos MRW / Zoom / Tealca / retiro.
- Configuración de tasa USD a Bs.
- Configuración de contacto y métodos de pago.
- Exportación de inventario CSV.
- Nota de entrega premium si está disponible en la versión base.

Importante:
Esta versión usa localStorage para pruebas en Netlify.
Cuando se conecte Supabase, el panel será real, persistente y accesible desde varios dispositivos.
