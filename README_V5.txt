ThinkStore V5 - Carrito + Registro + Nota de entrega

Nuevas funciones:
- Botón “Añadir al carrito” en cada producto.
- Registro de cliente con nombre, cédula/RIF, teléfono, correo, dirección, referencia y empresa de envío.
- Datos guardados localmente en el navegador del cliente.
- Generador de nota de entrega.
- Opción de copiar, imprimir o enviar la nota por WhatsApp.
- WhatsApp incluye producto, modelo, color, configuración, estado y datos del cliente.

Nota:
Esta versión funciona en Netlify como sitio estático. Para usuarios reales con contraseña y base de datos online,
la siguiente fase debe integrar Firebase o Supabase.
